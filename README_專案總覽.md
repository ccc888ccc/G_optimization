# 專案總覽索引（Optimization 教學網站）

一句話：做一個教「數學最佳化(optimization)」各領域的教學網站，靈感來自 ccjou《線性代數啟示錄》，但要現代化、有互動動畫、可持續更新。自學複習用，同時給大學部與研究所讀者。

## 目前資料夾內容一覽

**規劃與內容（我＝Claude 產出）**

- `內容大綱與規劃.md` — 總規劃：定位、三維度分類、領域大綱、學習路徑、寫作模板、視覺化清單、功能清單（忠實對照 ccjou）、技術方案（已定案）、部署（已定案）。
- `完整文章大綱.md` — 展開到單篇文章層級（約 110 篇），每篇標類型、重點、是否需動畫。
- `大綱編輯指示.txt` — 之後你用一句話下指令增修大綱的規則與句型。

**設計（Claude Design 產出，在 `網站設計交接/Optimization_LB/`）**

- `設計交接_給開發.txt` — 設計 token、徽章色值、頁面→元件對照、響應式斷點、技術落地備註。
- `G1 設計系統.dc.html` — 色票/字體/徽章/元件樣式預覽。
- `P2 文章頁公版.dc.html` — 核心長閱讀頁（KaTeX、可摺疊證明、互動容器、TOC、Giscus 佔位）。
- `頁面公版集.dc.html` — P1 首頁與 P3–P10 各頁公版＋響應式收合示範。
- `首頁三方向.dc.html` — 早期三方向探索（已定案 1a「書卷·學術期刊」，僅備查）。
- `support.js` — 上述 .dc.html 的預覽 runtime（設計預覽用，實作不需沿用）。

**部署**

- `部署方式.txt` — 定案：Astro 靜態站 + GitHub + Cloudflare Pages（全程免費，push 自動部署）。
