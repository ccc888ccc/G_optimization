## 這是什麼
數學最佳化（optimization）教學網站。靈感來自《線性代數啟示錄》：一篇講透一個概念、交叉引用、數學嚴謹，但現代化 + 互動動畫 + 手機可讀。
三支柱：**淺顯易懂 → 嚴謹證明（可摺疊）→ 動態視覺化**。視覺風格：**書卷／學術期刊**（暖色紙感、單一磚紅主色）。

- **repo**：https://github.com/ccc888ccc/G_optimization （private 可）
- **部署**：Cloudflare Pages（尚未接），先用 `*.pages.dev` 免費網址。

## 已定案決策
- **站名**：「G的呼吸小站」（副標 *Optimization, Illustrated*）→ 改站名只動 [`src/config.ts`](src/config.ts)。
- **徽章**：純文字 pill，**不加 icon**。
- **網域**：先用 pages.dev，暫不買。
- 標籤雲字級規則未定 → 程式內留 `TODO(G)`。

## 常用指令
```bash
npm install        # 首次安裝依賴
npm run dev        # 本機開發（http://localhost:4321）
npm run build      # 產出 dist/（驗證是否能建置）
npm run preview    # 預覽 build 結果
```

## 一鍵推送
雙擊 [`push.bat`](push.bat)，或：
```
push.bat "你的 commit 訊息"
```
- 不給訊息會提示輸入；直接 Enter 用時間戳。
- 已處理中文亂碼（UTF-8 + CRLF + chcp 65001）。origin 已設好，直接推。
